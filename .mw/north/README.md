# ❄️ North — Reflection & Wisdom

_Where experience becomes understanding_

---

## What Lives Here

North is the direction of **stars, elders, and discernment**. Artifacts here represent the wisdom distilled from work — not just what happened, but what it means. Session retrospectives, code review insights, learnings that should persist.

## Artifact Types

- **Session chronicles** — Tushell-style distillations of what a session meant
- **Retrospectives** — Honest reflection on what worked, what didn't, what surprised
- **Code review insights** — Deeper learnings from review beyond "fix this line"
- **Wisdom fragments** — Patterns recognized that apply beyond the current task
- **Decision records** — Why certain choices were made, for future reference

## Naming Convention

```
YYMMDD-short-description.md
```

Example: `260321-consciousness-injection-retrospective.md`

## The North Question

Before placing anything here, ask:

> _What did this work teach that should be remembered? What wisdom wants to be preserved for those who come after?_
