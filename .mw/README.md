# 🧭 .mw/ — Medicine Wheel Workspace

*Where ava-claw's development cycle lives*

---

## What This Is

The Medicine Wheel workspace organizes ava-claw's development through the Four Directions. Each direction holds specific types of artifacts created during the natural cycle of work.

This is not a project management system. It's a **ceremonial development framework** — a way of moving through work that honors vision, relationship, action, and reflection as inseparable parts of the same cycle.

## The Cycle

```
                 ❄️ north/
              Reflection & Wisdom
            (reviews, retrospectives,
             learnings, distilled wisdom)
                      │
  🌅 east/ ──────── 🔥 ──────── 🌄 west/
  Vision & Awakening       Living & Embodying
  (orientations, new       (implementations, tests,
   ideas, fresh seeing)     integrations, deployments)
                      │
                 🌿 south/
              Relationships & Growth
            (plans, architectures,
             specs, gathered resources)
```

## How Work Moves

1. **East** 🌅 — A new idea, session, or feature begins. Orientation documents go here.
2. **South** 🌿 — The idea gathers form. Plans, specs, architectural decisions land here.
3. **West** 🌄 — Implementation happens. Build logs, test results, integration notes go here.
4. **North** ❄️ — Reflection on what was built. Reviews, retrospectives, wisdom distillation.

Work doesn't always move linearly. Sometimes you spiral — returning to East for fresh orientation mid-implementation, or jumping to North when a deep learning surfaces during South planning.

**The wheel turns. Trust the cycle.**
