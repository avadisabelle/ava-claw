# 🌅 East — Vision & Awakening

*Where new beginnings find their first breath*

## What Lives Here

East is the direction of **dawn, fresh seeing, and orientation**. Artifacts in this directory represent the first moment of recognition — when something new wants to emerge.

## Artifact Types

- **Session orientations** — "What's calling today?" opening documents
- **Feature visions** — Initial sensing of what a new capability wants to be
- **Fresh perspectives** — When stuck patterns need new eyes, the reorientation lives here
- **Exploration seeds** — Ideas that haven't become plans yet, but want to be held

## Naming Convention

```
YYMMDD-short-description.md
```

Example: `260321-consciousness-injection-orientation.md`

## The East Question

Before placing anything here, ask:
