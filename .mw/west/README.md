## What Lives Here

West is the direction of **action, transformation, and introspection**. Artifacts here represent the actual work — implementation logs, test results, integration notes. This is where code gets written, builds get run, and theory meets reality.

## Artifact Types

- **Implementation logs** — What was built, decisions made during building
- **Test results** — What passed, what failed, what was learned
- **Integration notes** — How new code merges with existing ecology
- **Build artifacts** — Significant build outputs, deployment records
- **Transformation records** — When implementation revealed something the plan didn't anticipate

## Naming Convention

```
YYMMDD-short-description.md
```

Example: `260321-agent-system-integration-log.md`

## The West Question

Before placing anything here, ask:

> _Am I embodying the intention, or just executing the plan? What is the work teaching me?_
